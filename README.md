# ICADEM ULTRA PRO

Sitio web corporativo de una sola página listo para GitHub Pages.

## Estructura
- `index.html`
- `css/styles.css`
- `js/script.js`
- `assets/`

## Publicación en GitHub Pages
1. Sube el contenido del proyecto a un repositorio.
2. Activa GitHub Pages desde la rama principal.
3. Si usarás dominio propio, agrega tu archivo `CNAME`.

## Personalizaciones recomendadas
- correo oficial
- ciudad o cobertura
- métricas reales
- dominio y favicon final
